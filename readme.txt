== Description ==
Implemantace přihlašování, registrace a dalších služeb z informačního systému skautIS do WordPressu.

Plugin po aktivaci bude vyžadovat APP ID. Návod jak ho zístat najdete tady: https://napoveda.skaut.cz/skautis/skautis-integration

== Installation ==
1. Stáhnout si plugin a aktivovat
2. V levém menu se objeví položka skautIS
3. Musíte si zažádat o APP ID návod je tady: https://napoveda.skaut.cz/skautis/skautis-integration
4. Zadáte APP ID a plugin se plně aktivuje
